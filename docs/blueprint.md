# **App Name**: Family Fun Zone

## Core Features:

- Bowling Lane Booking: Allow users to select and book time slots for bowling lanes.
- AR Darts Reservation: Enable reservation of AR darts sessions.
- Soft Play Scheduling: Schedule access to the soft play area.
- Booking Management: Display upcoming bookings and allow cancelations.
- Activity Suggestions: Based on user age and booking history, this tool will suggest complementary activities that they might enjoy.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to evoke energy and fun. This color is chosen to align with a bright, modern aesthetic suitable for a family entertainment center.
- Background color: Light blue (#E5F5FB), a desaturated tone of the primary blue for a clean backdrop.
- Accent color: Energetic yellow (#F2E205) to highlight interactive elements and calls to action, differing in both brightness and saturation for maximum contrast.
- Font: 'Poppins', a geometric sans-serif for a contemporary and precise feel. It is suitable for headlines and shorter texts.
- Use playful and clear icons representing each activity (bowling, darts, soft play, etc.).
- A grid-based layout with rounded corners and clear spacing to make it user-friendly.
- Subtle animations for transitions and interactive elements to enhance the user experience.
